# Installation

## Requirements

- Python 3.10+

## Install from PyPI

```bash
pip install azure-functions-doctor
```

## Install from source

```bash
git clone https://github.com/yeongseon/azure-functions-doctor-for-python
cd azure-functions-doctor-for-python
python3 -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
```
